#Skin Disease Detection

An AI powdered web app that detects and classifies skin disease from Images 

## Features
- Upload Skin Image for Analysis
- ML Model Classifies the Disease
- Results Displayed Instantly

## Tech Steck
-Python
-Pytorch
-flask
-HTML/CSS

## How To Run
1.Clone The repo
2.Install Dependencies:'pip install -r requirements.txt'
3.Download Model from : [ https://drive.google.com/file/d/1oYWQk5OWaT4yzfz3ZTTFPtYqiLBLhxJh/view?usp=drive_link ].
4.Run: 'python app.py'
